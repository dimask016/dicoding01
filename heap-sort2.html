Apa itu Heap Sort?

Heap Sort adalah algoritma pengurutan yang memanfaatkan struktur data heap, yaitu struktur berbentuk complete binary tree.

üìä Prinsip Dasar Heap Sort
1. Membangun Heap:
Mengubah array menjadi struktur max-heap (parent node lebih besar dari child nodes) atau min-heap (parent node lebih kecil dari child nodes).
‚Ä¢ Untuk max-heap, elemen terbesar akan berada di root (indeks 0)
2. Proses Pengurutan:
Melakukan ekstraksi berulang-ulang hingga semua elemen terurut:
‚Ä¢ Menukar elemen terbesar (root) dengan elemen terakhir array
‚Ä¢ Memperkecil ukuran heap
‚Ä¢ Melakukan heapify pada root untuk mempertahankan sifat heap
‚Ä¢ Ulangi proses hingga semua elemen terurut

Jenis Heap:

Max-Heap
Setiap node parent memiliki nilai lebih besar atau sama dengan nilai anak-anaknya
Min-Heap
Setiap node parent memiliki nilai lebih kecil atau sama dengan nilai anak-anaknya
Contoh Max-Heap dari Array [3, 19, 1, 14, 8, 7]:
Array Setelah Build Max-Heap: [19, 14, 7, 3, 8, 1]

Pseudocode Heap Sort

Fungsi Utama: Heap Sort
PROCEDURE heapSort(arr)
    n = length(arr)
    // Build max heap
    FOR i FROM n/2 - 1 DOWN TO 0
        heapify(arr, n, i)
    
    // Extract elements from heap one by one
    FOR i FROM n-1 DOWN TO 1
        // Move current root to end
        SWAP arr[0] WITH arr[i]
        // Call heapify on reduced heap
        heapify(arr, i, 0)
Fungsi Pendukung: Heapify
PROCEDURE heapify(arr, n, i)
    largest = i          // Initialize largest as root
    left = 2*i + 1       // Left child
    right = 2*i + 2      // Right child
    
    // If left child is larger than root
    IF left < n AND arr[left] > arr[largest]
        largest = left
    
    // If right child is larger than current largest
    IF right < n AND arr[right] > arr[largest]
        largest = right
    
    // If largest is not root
    IF largest != i
        SWAP arr[i] WITH arr[largest]
        // Recursively heapify the affected sub-tree
        heapify(arr, n, largest)
Penjelasan Fungsi:
  • Heap Sort: Fungsi utama yang mengatur proses sorting.
  • heapify: Fungsi untuk mempertahankan sifat heap.
  • Build max heap: Mengubah array menjadi max-heap.
  • Extract elements: Mengambil elemen terbesar dan menempatkannya di akhir.

Summary Heapify & Leaf

üìä Summary Heapify
Heapify adalah proses untuk memastikan bahwa suatu node dan subtree-nya memenuhi aturan Max-Heap.
Inti proses Heapify:
1. Hitung anak kiri: left = 2*i + 1
2. Hitung anak kanan: right = 2*i + 2
3. Bandingkan parent dengan anak-anaknya
4. Jika salah satu anak lebih besar ‚Üí swap
5. Lanjutkan proses ke bawah (rekursif) sampai subtree kembali valid
Tujuan:
Menjaga struktur heap agar setiap parent selalu lebih besar dari anaknya dalam Max-Heap.
üçÉ Summary Leaf
Leaf (daun) adalah node pada heap yang tidak memiliki anak, yaitu node paling bawah.
Cara menentukan leaf:
Jika jumlah elemen array adalah n:
Leaf berada pada index:
index = floor(n/2) sampai n-1
Alasannya:
Karena pada array-based heap, node mulai dari floor(n/2) ke atas tidak punya child (keluar dari range array).
üîó Hubungan Leaf dan Heapify
Leaf tidak perlu di-heapify, karena tidak punya anak.
Heapify dimulai dari node bukan leaf, yaitu dari index floor(n/2) - 1 turun ke 0.
Inilah yang membentuk Build Max Heap.
Heapify Process
heapify(arr, n, i):
  largest = i
  left = 2*i + 1
  right = 2*i + 2
  if left < n and arr[left] > arr[largest]:
    largest = left
  if right < n and arr[right] > arr[largest]:
    largest = right
  if largest != i:
    swap(arr[i], arr[largest])
    heapify(arr, n, largest)
Leaf Identification
n = length(arr)
leaf_start = floor(n/2)

Example:
arr = [3, 19, 1, 14, 8, 7]
n = 6
leaf_start = floor(6/2) = 3
Leaf index: 3, 4, 5
Leaf values: 14, 8, 7

Implementasi Python untuk Hotel Makassar

import heapq

class Hotel:
    def __init__(self, name, price, rating, popularity_score, distance_from_center):
        self.name = name
        self.price = price
        self.rating = rating  # 1-5 stars
        self.popularity_score = popularity_score  # 1-100
        self.distance_from_center = distance_from_center  # in km

    def __lt__(self, other):
        """Untuk max-heap: hotel lebih prioritas harus 'lebih kecil' dalam min-heap"""
        # Priority 1: Rating (lebih tinggi = lebih prioritas)
        if self.rating != other.rating:
            return self.rating > other.rating  # Balik tanda untuk max-heap behavior
        # Priority 2: Popularity (lebih tinggi = lebih prioritas)
        if self.popularity_score != other.popularity_score:
            return self.popularity_score > other.popularity_score
        # Priority 3: Price (lebih murah = lebih prioritas)
        if self.price != other.price:
            return self.price < other.price
        # Priority 4: Distance (lebih dekat = lebih prioritas)
        return self.distance_from_center < other.distance_from_center

    def __str__(self):
        return f"{self.name} (Rating: {self.rating}, Price: ${self.price}, Popularity: {self.popularity_score}, Distance: {self.distance_from_center}km)"

class TravelokaSearchEngine:
    def __init__(self):
        self.hotels = []

    def add_hotel(self, hotel):
        # Gunakan min-heap dengan priority tuple
        priority_tuple = (
            -hotel.rating,  # Negasi untuk max-heap behavior
            -hotel.popularity_score,
            hotel.price,
            hotel.distance_from_center,
            hotel.name  # Untuk tie-breaker
        )
        heapq.heappush(self.hotels, (priority_tuple, hotel))

    def get_top_recommendations(self, k=5):
        recommendations = []
        temp_heap = self.hotels.copy()

        for _ in range(min(k, len(temp_heap))):
            if temp_heap:
                priority, hotel = heapq.heappop(temp_heap)
                recommendations.append(hotel)

        return recommendations

    def print_top_hotels(self, k=3):
        print(f"Top {k} Recommended Hotels di Makassar:")
        top_hotels = self.get_top_recommendations(k)
        for i, hotel in enumerate(top_hotels, 1):
            print(f"{i}. {hotel}")

# Contoh penggunaan dengan hotel-hotel di sekitar Makassar
def demo_traveloka_search():
    traveloka = TravelokaSearchEngine()

    print("Hotel tersedia di database Makassar:")
    hotels = [
        Hotel("Aston Makassar Hotel", 85, 4.5, 88, 1.2),
        Hotel("Swiss-Belhotel Harbour Bay", 120, 4.6, 92, 0.8),
        Hotel("Clarion Makassar", 75, 4.2, 82, 2.5),
        Hotel("Santika Premiere Makassar", 95, 4.4, 86, 1.5),
        Hotel("Four Points by Sheraton Makassar", 110, 4.7, 94, 1.0),
        Hotel("Losari Beach Hotel", 65, 4.0, 78, 3.0),
        Hotel("Hotel Makassar Golden", 50, 3.8, 72, 2.0),
        Hotel("Ibis Makassar City Center", 55, 3.9, 75, 1.8)
    ]

    for hotel in hotels:
        print(f"  - {hotel}")
        traveloka.add_hotel(hotel)

    print("\n" + "="*60)
    traveloka.print_top_hotels(5)

# Menjalankan demo
if __name__ == "__main__":
    demo_traveloka_search()
