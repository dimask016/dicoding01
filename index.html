Tentang Heap Sort

Heap Sort adalah algoritma pengurutan berbasis perbandingan yang menggunakan struktur data heap (binary heap). Algoritma ini bekerja dengan membangun max-heap dari array input, kemudian menukar elemen terbesar (root) dengan elemen terakhir, dan mengurangi ukuran heap secara berulang.

Kelebihan Heap Sort:

  • Kompleksitas waktu O(n log n) yang konsisten
  • Tidak membutuhkan memori tambahan (in-place)
  • Efisien untuk data besar
  • Cocok untuk priority queue
